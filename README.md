# YourBussiness - aplikacja do zarządzania klientami i procesami w firmie

Nasza aplikacja kierowana jest głównie do małych i dużych firm, które potrzebują elastycznego narzędzia do monitorowania, zarządzania i kontrolowania przepływu informacji i procesów w zespole i firmie. 

Aplikacja zawiera takie funkcje jak: 
- nowoczesny system CRM
- system monitorjący aktywnych pracowników
- monitorowanie postępu w zarządzaniu klientami
- śledzenie GPS pracownika
- kontrola dokumentacji
